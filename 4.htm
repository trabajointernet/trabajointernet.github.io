EL INTERNET
Un navegador web envía y recibe información (datos) de otras partes de la Web. La información recibida aparece en la ventana del navegador. Los datos se transfieren mediante el protocolo de transferencia de hipertexto (HTTP, por sus siglas en inglés), que es el método que controla cómo se comunica el texto, las imágenes y los vídeos en Internet. NAVEGADORES
Un navegador web es un programa que permite ver la información que contiene una página web. El navegador interpreta el código, HTML generalmente, en el que está escrita la página web y lo presenta en pantalla permitiendo al usuario interactuar con su contenido y navegar. 		Algunos tipos de navegadores son:
Internet Explorer, Microsoft Edge,
Chrome, Firefox,
Safari, Opera,
Brave, Browzar.
